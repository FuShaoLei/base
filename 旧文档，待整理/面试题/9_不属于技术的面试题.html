不属于技术的面试题

判断一个公司是否可去

  • 是否有自己的产品
  • 是否是双休

为什么从上家公司离职了

  • 疫情开放，怕感染到自己，所以离职回家了
  • 再加上家里有一些事情要处理

自我介绍

  1. 姓名，学校，什么时候毕业的
  2. 应聘什么职位
  3. 公司经历

自我介绍（旧）

  1. 姓名，学校，今年六月份能拿到毕业证
  2. 应聘什么职位，我有什么优势
  3. 大学学的就是android，老师教的比较基础，更多的一些知识都自学而来的，包括在大学里，也有和同学一起做小组的项目，积累的一定的经验（有十足的信心）
  4. 有在新浪实习过一段时间

你好，我叫符少磊，目前就读于河北师范大学，专业是软件工程，今年6月份毕业。因为对android开发很有兴趣，所以来应聘这份工作。我有着良好的java基础，熟悉面向对象思想，也有着良好的android基础，熟悉常用的布局，以及四大组件的使用。也曾经在新浪的新浪新闻安卓组里边实习过一段时间，主要负责的是信息流的模块，以及配合测试人员发现bug，解决bug。

以上就是我的自我介绍。

职业规划

  1. 对Android开发很感兴趣
  2. 在半年到一年的时间里，通过工作，摆脱学生的身份，稚气，不断的提升自己的技术，快速融入到职场中，成为一名比较合格的安卓程序员
  3. 然后此后的三到四年的时间里，提升自己的解决问题的能力，成为可以独挡一面的程序员，之后，更久远的职业规划，可能向底层深入（framework层），或者研究音视频方向的，这些都是有可能的
  4. 以上就是我的职业规划

薪资要求

  1. 非常欣赏贵公司，也很开心能有机会加入，但是我想先了解一下公司的制度在做决定
  2. 考勤制度
    1. 工作时间
    2. 如何考勤？指纹？钉钉？
    3. 有没有带薪休假
    4. 请假的话扣钱吗，固定的金额还是浮动的
  3. 薪酬结构
    1. 五险一金
    2. 实习期工资
    3. 有试用期吗？试用期工资 百分之几，还是全额
    4. 其他福利
    5. 有没有年终奖
    6. 发工资时间
  4. 询问工资
    1. 贵司对于这个职位预算的区间大概在多少到多少之内呢？
    2. 我期望薪资是
  5. 入职手续，需要准备什么吗

当确认薪资后要确认：

  1. 基本月薪 （不算补贴，绩效，提成的，基本月薪是否就是offer上写的工资）
  2. 社保和公积金（即五险一金）的缴纳基数是否就是基本月薪
  3. 公积金的缴纳比例是多少（5%~12%）

什么时候能到岗

我会在收到offer后，一到两周内到岗，最迟一个月内

对公司的期望

  • 希望公司有一个团结的团队能够一起工作共事
  • 希望学习的氛围，工作的氛围浓厚一些，不要搞一些办公室政治的有的没的东西
  • 希望在工作中很好的运用自己的知识和技能外，提高技术水平，与公司共同成长

如果录用你了，准备如何开展工作

  • 听从小组领导的安排，以及结合自身的情况 开展相应的工作
  • 最开始肯定是，看源码，了解总体的架构，不一定每个细节都要知道，至少了解过每一个模块是如何运行的

谈谈你的缺点

每个人都有一定的缺点，我的缺点是在解决不出问题的时候，倘若是只有我一个人的话，心情会低落，不过和大家在一起工作的话，应该不会出现这个问题

为什么选择我们公司呢

我十分看好贵司所在的行业，并且我认为贵司也十分注重人才，而且这项工作很适合我，我相信我一定能胜任这份工作的

兴趣爱好

没有什么特殊的兴趣爱好，平常看看书，看看电影，听听音乐之类的

反问

  • 公司的是否是自研？自研产品是？
  • 用到的技术栈
  • 大小周？还是双休？
  • 实习期工资
  • 试用期工资 百分之几，还是全额
  • 五险一金
  • 福利
  • 是否配备电脑
  • 三餐
  • 发工资时间
